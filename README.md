# Age Calculator

A simple web-based age calculator that allows users to input their birthdate and calculate their exact age in years, months, and days.

## Features

- Input birthdate using a date picker
- Calculates age in years, months, and days
- Validates input (prevents future dates)
- Responsive and user-friendly interface
- Built with vanilla JavaScript, no external dependencies

## How to Use

1. Clone or download the repository
2. Open `index.html` in your web browser
3. Select your birthdate from the date input
4. Click the "Calculate" button to see your age displayed
